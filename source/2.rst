新手必须掌握的Linux命令

2.1 查看帮助命令
---------------------

=========   ==============================
按键	     用处
空格键       向下翻一页
PaGe down	向下翻一页
PaGe up  	向上翻一页
home     	直接前往首页
end	        直接前往尾页
/	        从上至下搜索某个关键词，如“/linux”
?	        从下至上搜索某个关键词，如“?linux”
n	        定位到下一个搜索到的关键词
N	        定位到上一个搜索到的关键词
q	        退出帮助文档
=========   ==============================
